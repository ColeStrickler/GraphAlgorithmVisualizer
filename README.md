# GraphAlgorithmVisualizer
An application for visualizing graph algorithms. A*, Prim's, Kruskal's and more!


# Usage
Use spacebar to cycle through modes
- Mode 1: Place Source
- Mode 2: Place Destination
- Mode 3: Place Obstacle
